# Fourier Ptychography Datasets

## cos7-blue

Fixed Cos7 cells illuminated with the blue LED of the Adafruit 32x32 RGB LED matrix. This is a very small dataset intended for quick tests to make sure code is working, not for doing high quality reconstructions.

| Attribute | Value |
| --------- | ----- |
| Magnification | 10x |
| NA | 0.288 |
| Pixel size | 5.86 um x 5.86 um |
| LED matrix axial offset | 50 mm |
| LED matrix pitch | 4 mm x 4 mm |
| LED matrix rotation | 90 degrees |
